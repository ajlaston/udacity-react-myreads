## MyReads Book tracking app powered by React

## Author : Adam Laston

## Description :

This Application has a database of books that you can look through using
by using the search function that track if you are currently reading, want 
to read or have read the book.

## Installation Instructions:

Download or fork the project and install dependencies by typing
'npm install' in your terminal of choice.

 once everything is installed type 'npm start' to start the application.

 2.CSS
